## About the Project

bsn.js is a specialized wrapper for Bootstrap modal dialogs.
It is designed to provide functionality similar to native alert(), confirm(), and prompt() methods, but with a more stable, customizable, and responsive user experience.
Unlike standard browser dialogs, this plugin offers a seamless interface that integrates perfectly with your web application's design while maintaining technical stability.

---

## Accessibility-First

Attention: This plugin is specifically optimized for visually impaired users and screen reading software. It ensures proper focus management and ARIA attributes, making interactive notifications accessible to everyone.

---
 
## Quick Start

To use the plugin, simply include the script in your project:

```html
<script src="./src/bsn.js"></script>
```

All methods are available through the global $gn.bsn object.

--- 

## Features


- Lightweight: Minimal overhead on your project.
- No Manual HTML: Dialog structures are generated automatically.
- Async Support: Works perfectly with modern JavaScript (async/await).
- Multi-language: Ready for localization (RU, UA, EN, DE).
