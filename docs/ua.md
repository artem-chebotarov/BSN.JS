## Опис проекту

bsn.js (Bootstrap Service Notification) — це легка обгортка над діалоговими вікнами Bootstrap. Плагін створений для заміни стандартних браузерних функцій alert, confirm та prompt на більш сучасні та гнучкі інтерфейси.
Головний пріоритет: Повна підтримка програм екранного доступу (Screen Readers). Це означає, що незрячі користувачі не матимуть жодних проблем під час взаємодії з вашими діалогами.

---

## Підключення

Щоб почати роботу, просто додайте скрипт на вашу HTML-сторінку:

```html
<script src="./src/bsn.js"></script>
```

### Увага!

Після підключення плагіна в проекті з'являється глобальна змінна $gn. Цей формат походить з GiannaFramework. Тепер у $gn.bsn зберігається посилання на всі функції сповіщень. Пам’ятайте: якщо ви використовуєте інші модулі GiannaFramework, вони також будуть доступні через об’єкт $gn.*.

## Методи API

### 1. $gn.bsn.alert()

Метод дозволяє надіслати важливе сповіщення, яке неможливо ігнорувати.

#### Приклад виклику:

```js
$gn.bsn.alert({
    title: "",    // Заголовок вікна
    message: "",  // Текст повідомлення
    ok_text: "",  // Напис на кнопці (наприклад, "Зрозуміло")
    app: "",      // Додатковий підпис програми (наприклад, "system-logger")
    ok: () => {}  // Callback, який виконується після натискання кнопки
});
```

Цей метод може повернути результат з callback або undefined.


---

### 2. $gn.bsn.confirm()

Метод запитує підтвердження дії. Ви також можете призначити callback-функції на кнопки. Метод повертає логічне значення (boolean).

#### Приклад виклику:

```js
let result = await $gn.bsn.confirm({
    title: "",    // Заголовок
    message: "",  // Текст запитання
    app: "",      // Підпис скрипта або модуля
    ok_text: "",  // Текст кнопки підтвердження
    no_text: "",  // Текст кнопки скасування
    ok: () => {}, // Callback на підтвердження
    no: () => {}  // Callback на відмову від дії
});
 ```

 ---

### 3. $gn.bsn.prompt()

Метод запитує введення даних у користувача. Повертає введене значення або undefined. Можна призначити callback на підтвердження або скасування.

#### Приклад виклику:

```js
let value = await $gn.bsn.prompt({
    title: "",    // Заголовок
    message: "",  // Опис того, що потрібно ввести
    input: "",    // Назва (title) для поля введення (важливо для доступності)
    value: "",    // Значення в полі за замовчуванням
    app: "",      // Підпис програми
    ok_text: "",  // Текст кнопки підтвердження
    no_text: "",  // Текст кнопки скасування
    ok: () => {}, // Callback підтвердження
    no: () => {}  // Callback скасування
});
```

---

### 4. $gn.bsn.toast()

Метод виводить спливаючі toast-сповіщення, які зникають автоматично.

#### Приклад виклику:

```js
$gn.bsn.toast({
    title: "",    // Заголовок
    message: "",  // Текст повідомлення
    type: "",     // Колір/тип (наприклад: success, danger, warning, info)
    delay: 5000,  // Час показу в мілісекундах
    ok_text: "",  // Напис на кнопці всередині сповіщення
    ok: () => {}  // Callback для кнопки
});
 ```

## Технічні примітки

- Вам не потрібно створювати HTML-шаблони для вікон вручну. Вони вже інтегровані в клас.
- Шаблон кожного методу додається на сторінку лише один раз. Після цього клас просто перевіряє наявність шаблону та оновлює його дані.
- Бібліотека не засмічує DOM-дерево зайвими елементами.
- Можна не вказувати всі параметри конфігурації — у такому разі будуть використані стандартні тексти англійською мовою.
- Кожен елемент інтерфейсу оптимізований для того, щоб скринрідери коректно передавали фокус та озвучували контент.
